== Support for Google Cloud NetApp Volumes

Ransomware Resilience now supports Google Cloud NetApp Volumes systems, enabling you to efficiently detect and respond to ransomware threats in Google Cloud NetApp Volumes. When you discover workloads, Ransomware Resilience now presents Google Cloud NetApp Volumes and displays them in the protection dashboard. Ransomware Resilience support for Google Cloud NetApp Volumes includes detection and protection strategies with snapshots only. Support for Google Cloud NetApp Volumes is currently in preview.
